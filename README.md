### Building

You should use Maven to build the plugin.
```
mvn clean install
```

In IntelliJ IDEA, you can run the tasks from the Maven menu on the right side.

### Contributing

To contribute, feel free to create your own fork repository.  
You can then issue a merge request towards this project once your changes are made.
